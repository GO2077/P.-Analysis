This repository contains some of the work done on analyzing genetic, epigenetic, and transcription data obtained through blood samples. 

Functions and plate_check_final were used to handle incorrectly labeled plates where the participant IDs did not match the data assigned to them. 
For testing purposes functions contains some manually written functions, instead of pre-existing ones. 

Analysis_test and PCA contain the start of the analysis of the methylation data. These will be added to as the EWAS is performed.
